HTML é uma linguagem de marcação utilizada na construção de páginas na Web.
Documentos HTML podem ser interpretados por navegadores.

HTML é a abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de
Marcação de Hipertexto.

Abaixo, temos o exemplo de duas listas de itens:

Linguagens utilizadas para programação web:
  • HTML
  • CSS
  • JavaScript
  • PHP
Tecnologias para desenvolvimento mobile:
  • Flutter
  • React Native
  • Ionic

Passos para aprender a programar:

  • Estudar bastante as linguagens necesárias.
  • Fazer muitos exercícios.
  • Elaborar vários projetos.
  • Buscar oportunidades no mercado do trabalho.
